# Order Service

Order Service is an application impleneted using SAP CAP, SAP HANA integrating with Java/Spring Boot.

## Running project loccally

Use the package manager [maven](https://maven.apache.org/) to to run locally.

```bash
# maven
mvn spring-boot:run
```
